# Manual Test Artifacts

This directory stores encrypted workbook outputs generated by `dotnet test`.

Generated files:

- `simple_image_en.xlsx`
- `japanese_en.xlsx`
- `simple_en.xlsx`
- `excel_sample_en.xlsm`
- `simple_image_ja.xlsx`
- `japanese_ja.xls`
- `simple_ja.xlsx`
- `excel_sample_ja.xlsm`

To regenerate them, run:

```bash
dotnet test tests/ExcelEncryptor.Tests/ExcelEncryptor.Tests.csproj --filter Generate_ManualTestFiles_WritesEncryptedWorkbooks
```
